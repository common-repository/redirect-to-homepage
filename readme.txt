Customize your login form page and redirect to homepage

== Description ==
Redirects to home page then a user will login into dashboard and admin bar will appear in top.

== Installation ==

1. Extract zip in the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. No need to do anything

== Frequently Asked Questions ==
= Why i need this plugin? =
Because to reduce the time of visiting any users and make some interesting to redirect your wordpress blog/website in your homepage neither in the dashboard. It'll make fun to see the admin bar in the top. 
= Do i need to anything else to use this plugin? =
No, you don't have to do anything but just activate the plugin and see the result. First activate the plugin and logout from your admin panel and again login into your admin bar. That's it.
